# PR #94 变更总览

- **聊天**：渲染链路稳定化、工具事件可视化、聊天虚拟滚动与滚动行为重构。
- **Agent**：管理与展示优化，多 Agent 渠道绑定与能力扩展。

- **安装/部署**：独立安装包、镜像源切换、Docker/CLI 诊断与引导增强。
- **诊断/修复**：Gateway 一键补丁、自动检测与修复路径完善。
- **性能**：ARM 设备优化、缓存与轮询降频、R2 分发策略与通用包。

## 3) 质量与安全
